MARCO TEÓRICO

Fundamentos Biomecánicos y Movimiento Humano

Mecánica del Sistema Musculoesquelético

La biomecánica aplica la mecánica clásica para analizar fuerzas, torques y movimiento en contextos estáticos y dinámicos.

Planos y Ejes Anatómicos

El análisis se organiza en planos sagital, frontal y transversal, permitiendo describir flexión, extensión, abducción, aducción y rotaciones.

Grados de Libertad Articular

Las articulaciones sinoviales definen qué combinaciones de movimiento son posibles: hombro (3 ejes), codo (1 eje), rodilla (1 eje principal).

Rango de Movimiento Articular (ROM) y Goniometría

Definición de ROM

Amplitud angular máxima que una articulación puede alcanzar de forma activa o pasiva dentro de sus límites fisiológicos. Es un indicador básico de movilidad funcional.

Valores de Referencia por Articulación:

Articulación Movimiento ROM Normal
Hombro Flexión / Abducción 180° / 180°
Codo Flexión 145°
Cadera Flexión / Abducción 120° / 45°
Rodilla Flexión 135°
Tobillo Dorsiflexión / Plantar 20° / 50°

Fuente: Norkin & White, 2016; Levangie & Norkin, 2011

️ Desafíos Técnicos Resueltos

1. Integración Hardware-Software en Tiempo Real

Problema: Sincronizar cámara (30 FPS), MediaPipe, interfaz web y control Bluetooth sin lag
Solución: Multi-threading con 4 hilos independientes + Singleton para MediaPipe compartido

2. Cálculo Angular Preciso desde Landmarks 3D

Problema: Landmarks con ruido y coordenadas normalizadas variables
Solución: Ley del coseno con vectores + Percentil 95 para eliminar outliers

3. Posicionamiento Repetible de la Cámara

Problema: Cada usuario tiene estatura diferente → diferentes alturas de articulación
Solución: Proporciones Drillis-Contini + motor con encoder (precisión ±0.1mm)

4. Comunicación Inalámbrica Confiable en Paneo

Problema: Bluetooth puede tener latencia variable y pérdida de paquetes
Solución: Protocolo con comandos simples (L/R/C/S) + confirmación de estado

Resultado: Sistema funcional con R² = 0.996 vs goniómetro manual y bias +1.26°

CONCLUSIONES Y RECOMENDACIONES

Conclusiones Principales

OBJ. GRAL

Se desarrolló un sistema de análisis biomecánico monoplano basado en visión artificial y cámara RGB, capaz de estimar ROM con fines didácticos. Integra adquisición, procesamiento de puntos anatómicos y visualización en tiempo real.

OBJ. 1-2

La arquitectura modular (adquisición → cálculo → visualización → soporte físico) estableció un flujo operativo claro: configurar–capturar–validar–guardar, mejorando la trazabilidad y replicabilidad del procedimiento práctico.

OBJ. 3-4

La comparación con goniómetro mostró R² ≈ 0.996, sesgo medio de 1–2° y error < 5° en hombro, codo, cadera y rodilla. El tobillo (~6°) se identificó como caso prioritario de mejora.

OBJ. 5

El sistema reduce variabilidad manual, estandariza la evaluación de ROM y documenta resultados por estudiante/ejercicio. Alta aceptación de docente y estudiantes (>95% motivación, 81.3% desea continuar).

BioTrack es una herramienta didáctica funcional para la práctica de medición de ROM, análisis comparativo y discusión de fuentes de error en biomecánica educativa.

️ Limitaciones

Sistema 2D monoplano

Muestra limitada

~135 mediciones, sin estudio formal de repetibilidad con ICC

Referencia manual

Goniómetro manual como patrón (tiene su propio error)

Alcances

Contexto educativo

Importante: El sistema no reemplaza sistemas ópticos multicámara ni es apto para decisiones clínicas finas. Está diseñado específicamente para el contexto docente.

Recomendaciones: Ajustes Inmediatos

SOP de Captura Estandarizado

  • Distancia y altura de cámara con marcas físicas
  • Plano sagital/coronal definido con cintas en piso
  • Guía breve documentada para docentes

Filtro de Calidad de Pose

  • Umbrales mínimos de confianza de landmarks
  • Verificación de alineación con plano
  • Semáforo visual + solicitud de retoma

Protocolo Específico: Tobillo

  • Marcas visibles para talón y punta del pie
  • Métrica auxiliar tibia–calcáneo–eje pie
  • Restricción a rango fisiológico del ejercicio

Checklist Pre-Práctica (~1 min)

  • Verificar iluminación uniforme
  • Fondo no distractor
  • Vestimenta con contraste (no holgada)

Recomendaciones: Trabajo Futuro

Estudio de Repetibilidad y Fiabilidad

  • 10-15 repeticiones por articulación
  • Métricas: MAE, RMSE, IC 95%
  • Índices ICC intra- e inter-observador
